## 📖 About The Project

Terminal Dev is a modern web application inspired by developer terminals and command-line interfaces. The project showcases clean UI design, responsive layouts, and modern frontend development practices using React and TypeScript.

The goal of this project is to combine aesthetics and functionality while demonstrating strong frontend development skills and professional coding standards.

### Clone the Repository

```bash
git clone https://github.com/Rritesh005/terminal-dev.git
```

### Navigate to Project Directory

```bash
cd terminal-dev
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
